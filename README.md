# Inventory Management System

A modern, React-based inventory management system for in-house usage with authentication, multi-department tracking, and comprehensive reporting.

## 🚀 Features

- **Authentication**: Secure login/signup with Supabase Auth
- **Multi-Department Support**: Track draws by 11 departments
- **Overdraw Capability**: Allow negative inventory with back-order tracking
- **Flexible Units**: Support for EA, Dozen, Pair, etc. with automatic conversion
- **Image Support**: Attach and preview item images
- **Real-time Dashboard**: Multi-item drawing interface
- **Advanced Reporting**: Date filtering, CSV export, analytics
- **Responsive Design**: Works on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **Frontend**: React 18, Tailwind CSS, Heroicons
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Hosting**: Netlify
- **Version Control**: GitHub
- **CI/CD**: GitHub Actions

## 📋 Prerequisites

- Node.js 18+ and npm
- Supabase account
- Netlify account
- GitHub account
